# Skills Syllabus

Public, semi-packed AI skill artifacts generated from source skills in `/mnt/ops/prj/skills/dev`.

## Source Of Truth

- Edit source skills in: `dev/<skill-name>/`
- Do not manually edit packaged skills in this repository.

## Build Flow

Skills are generated by:

```bash
python3 dev/skill-deploy/scripts/skill_deploy.py dev/<skill-name>
python3 dev/skill-deploy/scripts/skill_deploy.py --all
```

The deploy flow writes:

- Public artifacts here (`syllabus/`)
- Runtime artifacts to `.agent/skills/`
